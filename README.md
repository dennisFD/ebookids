# Dunia Buku Anak

## Cara Menjalankan

Install dependency:

```bash
npm install
```

Jalankan project:

```bash
npm run dev
```

Build project:

```bash
npm run build
```

## Menambahkan PDF

Masukkan file PDF ke folder:

```bash
public/buku-anak.pdf
```

## Deploy ke Vercel

1. Upload project ke GitHub
2. Import repository di https://vercel.com
3. Framework: Vite
4. Build Command: npm run build
5. Output Directory: dist
